优雅的开源短URL生成API，现在，回应您的等待。

Cinnamoroll Dwz是一个用PHP和HTML编写的开源项目，这意味着您可以在自己的web服务器上自由的部署它！

## 特征

- 一个功能齐全的短链接API服务
- 无需登录，免费使用
- 在线与朋友分享API！
- 易于使用

## 要求

- 靠谱的Web服务器 (Nginx,Apache or IIS)
- PHP >= 7.0.0
